## 2. API Endpoints for Integration

### Admin Management
- **View Settings**: `GET /api/admin/settings` (See active email and current template).
- **Update Settings**: `POST /api/admin/settings`
  *Payload Example:*
  ```json
  {
    "active_smtp_user": "support@jabooks.shop",
    "from_name": "James Books Support",
    "otp_subject": "Your Security Code",
    "otp_body_template": "Your OTP is {otp}. Check your SPAM folder if you don't see it!",
    "smtp_host": "mail.privateemail.com"
  }
  ```

### Universal Data Storage (Products, Orders, Users)
- **Save Anything**: `POST /v1/data/{table_name}`
  *Example for Products:* `POST /v1/data/products`
  *Example for Orders:* `POST /v1/data/orders`
- **Retrieve Anything**: `GET /v1/data/{table_name}`
