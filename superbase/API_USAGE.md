# Superbase DBaaS API Usage Guide

## Authentication
`POST /auth/signup`
`POST /auth/verify-otp`
`POST /auth/login`

## Data API
`POST /v1/data/{table}`
`GET /v1/data/{table}`
`PATCH /v1/data/{table}/{id}`
`DELETE /v1/data/{table}/{id}`

## Dashboard
Visit `http://localhost:7777/dashboard` to view logs and stats.
