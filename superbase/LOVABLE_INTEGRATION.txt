# Superbase DBaaS Integration Guide for Lovable

## 1. API Configuration
- **Base URL:** http://localhost:7777
- **Port:** 7777 (Live)

## 2. Authentication Flow

### Step A: Signup (Request OTP)
**Endpoint:** `POST /auth/signup`
**Payload:**
```json
{
  "email": "user@example.com",
  "password": "strongpassword"
}
```

### Step B: Verify OTP
**Endpoint:** `POST /auth/verify-otp`
**Payload:**
```json
{
  "email": "user@example.com",
  "otp": "123456"
}
```

### Step C: Login (Get API Key)
**Endpoint:** `POST /auth/login`
**Payload:**
```json
{
  "email": "user@example.com",
  "password": "strongpassword"
}
```
**Response:** `{"api_key": "db_..."}`

---

## 3. Data API Usage
Once logged in, use your `api_key` in the Authorization header:
- **Header:** `Authorization: Bearer db_your_api_key_here`
- **Insert Data:** `POST /v1/data/{table_name}`
- **Fetch Data:** `GET /v1/data/{table_name}`
